# Взамєодія між рівнями

## Взамєодія LVL0 та LVL1
LVL0 в даній версії PFw відсутній як окремий обєкт. LVL1 взамєодіє з IO PLC через відповідні входи/виходи та статусні діагностичні біти. 

## Взамєодія LVL1 та LVL2
LVL2 підкорює LVL1 і відповідає за його конфігурування. Усі незадіяні змінні підчиняються PLC.
Конфігурування LVL1 через HMI реалізоване через конфігурування LVL2. Також доступний механізм конфігурування через буфери VAR, але цей механізм передбачається не через SCADA/HMI.

## Дії при форсуванні
При форсуванні дискретних змінних значення змінюється одним з наступних варіантів:
- командами з буферу
- зміною бітів в HMI.STA
- командами з володаря CM LVL2, який змінює значення через STA (`STA.SET=TRUE` на один цикл)

При форсуванні аналогових змінних значення змінюється одним з наступних варіантів:
- командами з буферу MIN/MAX
- зміною HMI.V в SCADA 
- командами MIN/MAX з володаря CM LVL2, який змінює значення через `CFG.V`(`STA.SET=TRUE` на один цикл)  
- зміною значення через `CFG.V`(`STA.SET=TRUE` на один цикл)
